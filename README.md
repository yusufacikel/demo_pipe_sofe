# Project Setup
Requires Python version 3.9, 3.10, 3.11, or 3.12.

Clone the repository:
```bash
git clone https://github.com/yusufacikel/demo_pipe_sofe.git
```
Change directory:
```bash
cd demo_pipe_sofe
```
## For Linux users:
Make the setup script executable and run it:
```bash
chmod +x setup.sh
./setup.sh
```
## For Windows users:
Run the batch setup script:
```bash
.\setup.bat
```
